Courses

Basic Web Dev

This course will take you from zero to web hero, so no prior experience with HTML or CSS necessary! You'll learn the most important aspects of both HTML and CSS as you build a complete 5 page website from the ground up. Along the way, you'll see how to code cutting-edge great looking sites as well as tried and true techniques. $500 - Enroll now for course starting March 1st, 2019!
  • HTML Tags & attributes
  • CSS Properties & values
  • Smooth Transitions
  • Transformations
  • Animations

Mobile Web Dev

This course is designed as direct follow-up to the Basic web development course, but is open to anyone with an existing, solid knowledge of HTML & CSS. The primary topic of this course is learning how to make websites that perform and look great on mobile devices AND desktop devices. You'll start by taking the project from the basic course and getting it to look great on mobile without affecting the desktop version. $500 - Enrollment Openson March 1, 2019.
  • Mobile Design
  • Media Queries
  • Breakpoints
  • Orientations
  • Mobile Detection

About

Our Story

Interface Web School Is Located in Omaha, Nebraska, and has been teaching web technology courses live nd in-person since 2014. As a start-up, we were free to develop our personal style of instruction that emphasizes the practical topics AND experiences to help foster a deeper understadning of tech that goes beyond just the syntax.

In 2017 Interface School was acquired by the AIM Institute. Aim's mission is to grow, connect and inspire the tech talent community through educational and career development initiatives. Our focus is to get people excited about technology and inspire them to pursue a tech career - which leads to abetter work force and ultimately lets communities to thrive.

FAQ

Courses

Do I need experience?
For the basics of Web Development Course, no - this course will teach you everything you need to know. For the mobile Web Dev Course, yes - you should have a solid understanding of HTML & CSS, including using advanced selectors like nth-of-type and :target.
DO I need to install any software?
You only need Chrome - everythingwill be presented online, and you will use our online development platform for all our coding work.
Is the course hard?
Maybe - kinda depends on you! The material is presented in a very step by step manner as it guides you through new topics while you are developing a real website. Remember that these are also self-paced courses, so if you need to review a certain section multipple times to really "get it" you can.
How long is the course?
Courses are open from the first of the month for 30 days, and each course is divided into 8-10 sections, so you typically have about 3 days on average to complete a section. Each section can take anywhere from 1-4 hours to complete.
